Paranuara Challenge

A set of APIs to query data about Paranuara's companies and citizens.

  • Given a company, the API needs to return all their employees.

    api/v1/company/:company_id/

  • Given 2 people, provide their information (Name, Age, Address, Phone) and the list of their friends in common which have brown eyes and are still alive.

    api/v1/person/:id1/:id2/

  • Given 1 people, provide a list of fruits and vegetables they like.

    api/v1/person/:id/food/
